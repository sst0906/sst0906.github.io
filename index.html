云计算实验报告

1. 实验目的

本次实验旨在利用 GitHub 搭建个人主页，以了解云计算环境下的静态网页托管与版本控制。

2. 实验步骤与内容

  1. 注册 GitHub 账号。
  2. 创建一个新的仓库用于托管个人主页，仓库名为 sst0906.github.io
  3. 使用 HTML、CSS 或其他前端技术编写个人主页内容。
  4. 将编写好的代码上传至 GitHub 仓库。
  5. 在仓库的设置中启用 GitHub Pages 功能，选择从主分支（通常是 master 分支）构建站点。
  6. 访问 https://sst0906.github.io/，查看您的个人主页。

3. 结论分析与体会

通过本次实验，我成功地搭建了自己的个人主页，并将其托管在 GitHub 上。在这个过程中，我学会了如何使用 GitHub 进行版本控制，以及如何利用 GitHub Pages 功能进行静态网页托管。这对于未来在云计算环境下开发和部署网页应用具有重要意义。

此外，通过编写个人主页的过程，我对前端技术有了更深入的了解，并提升了自己的网页设计与开发能力。

总的来说，本次实验使我对云计算和前端开发有了更加全面的认识，为我未来的学习和工作奠定了基础。
